The Ibis bug tracking system is available through the Ibis web-site:
http://www.cs.vu.nl/ibis. If you cannot for some reason report a bug
through this system, please Email your report to ceriel@cs.vu.nl.

Before  reporting  a bug, please make sure that you have the latest
Ibis distribution (check with the Ibis website). Also check with
the known bugs section in the Ibis website that it has not been reported
before (see the bugtracking page).
Please supply the output, and also supply the reason why you think
that it represents a bug. Don't make us guess.
Give the complete command line that triggers the bug, and supply
everything we need to try and reproduce the bug. Also please specify
which Java engine you used.
